如何申請 Google OAuth 2.0 Client ID 和 Client Secret

感謝您使用我們的應用程式！為了使用 Google 的 API，您需要申請自己的 Client ID 和 Client Secret。請按照以下步驟操作：

步驟 1: 創建 Google Cloud Project

  1. 訪問 Google Cloud Console。
  2. 登錄您的 Google 帳戶。
  3. 點擊左上角的「選擇專案」，然後點擊「新建專案」。
  4. 為您的專案命名，然後點擊「創建」。

步驟 2: 啟用 API

  1. 在專案中，導航到「API 和服務」>「庫」。
  2. 啟用您需要的 API，例如 Google People API 或 Google Drive API。

步驟 3: 設置 OAuth 2.0 憑證

  1. 在「API 和服務」>「憑證」中，點擊「創建憑證」並選擇「OAuth 客戶端 ID」。
  2. 選擇應用程式類型（例如 Web 應用程式）。
  3. 設置授權的重定向 URI，例如 http://localhost:8080/oauth2callback。
  4. 點擊「創建」。

步驟 4: 獲取 Client ID 和 Client Secret

創建憑證後，您將獲得 Client ID 和 Client Secret。請妥善保存這些信息，因為您將在應用程式中使用它們。

步驟 5: 設置環境變數

在您的系統中設置環境變數，以便應用程式可以讀取這些憑證：

# 在 Windows 中
set GOOGLE_CLIENT_ID=your_client_id_here
set GOOGLE_CLIENT_SECRET=your_client_secret_here

# 在 Linux 或 macOS 中
export GOOGLE_CLIENT_ID=your_client_id_here
export GOOGLE_CLIENT_SECRET=your_client_secret_here

完成

現在您已經成功申請了自己的 Google OAuth 2.0 Client ID 和 Client Secret，您可以使用這些憑證來運行我們的應用程式！
